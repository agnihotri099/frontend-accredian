# Installation

1. Clone the repository

For HTTP

```sh
git clone https://github.com/agnihotri099/frontend-accredian
```

Backend repo:

```sh
git clone https://github.com/agnihotri099/backend-accredian
```

2. Install dependencies

```sh
 npm i
```

3. Run server

```sh
npm run start
```

4. Run frontend

```sh
npm run start
```

# Environment Variables

```sh
  host: "localhost",
  user: "",
  password: "",
  database: "",
```
